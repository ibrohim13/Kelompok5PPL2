GET /students - Untuk mendapatkan daftar semua siswa.
GET /students/X:id - Untuk mendapatkan detail siswa berdasarkan ID.
POST /students - Untuk menambahkan siswa baru. Pastikan untuk mengirim data siswa dalam format JSON di body request.
PUT /students/X:id - Untuk memperbarui data siswa berdasarkan ID. Kirim data siswa yang diperbarui dalam format JSON di body request.
DELETE /students/X:id - Untuk menghapus siswa berdasarkan ID.

NOTE: UBAH "X" KE ID ANGKA TERLEBIH DAHULU
